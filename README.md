# PGT & HO Pay Justice Pakistan 🇵🇰
## #PGTPayJustice #HOPayJustice — Campaign Website

**Live site (once deployed):** https://pgthopayjustice.github.io

A grassroots advocacy platform for federal Postgraduate Medical Trainees (PGTs) and House Officers (HOs) whose stipends have been frozen since **27 January 2020** — with Federal Budget 2026–27 once again containing **zero allocation** for revision (7th consecutive year).

---

## 📦 What's in This Package

| File | Purpose |
|------|---------|
| `index.html` | The complete website — single file, all CSS/JS inline, no external dependencies except Google Fonts |
| `admin.html` | Admin panel for managing signatures, parliamentarians, and content (login: `admin` / `PGT@Justice2026`) |
| `docs/README.txt` | Instructions for uploading your scanned official notification images |
| `.nojekyll` | Required file — tells GitHub Pages to serve the HTML directly |

---

## 🚀 How to Upload to Your Live Site

You already have your GitHub account and repository set up. Now:

1. Go to your repository: `github.com/pgthopayjustice/pgthopayjustice.github.io`
2. Click **"Add file"** → **"Upload files"**
3. Drag in `index.html`, `admin.html`, and `.nojekyll`
4. Create a folder called `docs` and upload `docs/README.txt` into it
5. Scroll down, click **"Commit changes"**
6. Wait 2–3 minutes, then visit `https://pgthopayjustice.github.io`

If the site was already live, this upload **overwrites and replaces** the previous version with all the new sections (House Officer data, petition, stipend reporter, complaint center, parliamentarian contacts, etc).

## 🔒 Content Protection

The site implements right-click blocking, keyboard shortcut blocking (Ctrl+P/S/C/A, F12, PrintScreen), text selection disabling, drag prevention, and a print CSS block. No protection is 100% foolproof — these measures deter casual copying.
